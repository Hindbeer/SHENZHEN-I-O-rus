* *tag-reader* — это неблокируемый вход XBus, подключенный к считывателю RFID-меток.
* *food-a* и *food-b* являются простыми выходами, подключенными к дозаторам корма для домашних животных.
* Автоматическая кормушка для домашних животных может быть настроена для использования до трех домашних животных с RFID-метками, установленными на ошейнике, которые хранят значение 1, 2 или 3. Для каждого домашнего животного пользователь может указать минимальное количество единиц времени между кормлениями (ВРЕМЯ/TIME), количество корма, которое необходимо выдавать (РАЗМЕР/SIZE), и тип корма, который необходимо выдавать (ТИП/TYPE).
* Настройки конфигурации задаются пользователем с помощью комбинации дисков, которые могут считываться как входы XBus, и переключателей, которые могут считываться как простые входы.
