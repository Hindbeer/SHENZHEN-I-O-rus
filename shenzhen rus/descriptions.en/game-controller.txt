* *x* и *y* — простые входы, подключенные к джойстику.
* *a* и *b* — простые входы, подключенные к кнопкам.
* *radio-rx* — неблокирующий вход XBus, подключенный к радиоприемнику.
* *radio-tx* — выход XBus, подключенный к радиопередатчику.
* Когда пакет данных принимается по радио от игровой консоли, ответьте следующим пакетом данных из 3 значений, убедившись, что используются самые последние значения ввода:

#############

|     |     | 0, если *a* = 0 и *b* = 0
| *x* | *y* | 1, если *a* = 100 и *b* = 0
|     |     | 2, если *a* = 0 и *b* = 100
|     |     | 3, если *a* = 100 и *b* = 100
